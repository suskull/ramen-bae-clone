## Key Components

### 1. Edge Function
- Handles HTTP requests
- Authenticates users
- Processes templates
- Sends via Resend API
- Logs to database

### 2. Resend API
- Receives email payload
- Validates content
- Delivers email
- Returns email ID
- Tracks delivery

### 3. Database
- Stores email logs
- Tracks delivery status
- Enables history queries
- Supports analytics

### 4. Frontend
- Collects email data
- Calls Edge Function
- Displays results
- Shows history
